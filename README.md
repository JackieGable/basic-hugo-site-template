## Basic Hugo Site Template

1. Clone the repo:
```
git clone https://github.com/
```
2. Install bootstrap:
```
npm install bootstrap@v5.3.2
```
3. Run the hugo server:
```
hugo serve
```
4. Open the page in the browser:  
http://localhost:1313/

5. Initialize a repo (use GitBash)
``` git init ```
